# Crypto Detective 8-bit Prototype

This folder keeps the 8-bit RPG version separate from the main web CTF.

## Pages

- `index.html` - instructor briefing handoff + 15-question quiz gate
- `storyboard.html` - interactive storyboard / pitch
- `game.html` - first playable prototype

## Controls

- `WASD` or arrow keys: move
- `E`: interact / collect evidence
- `Enter`: close dialogue

## Current Prototype

- instructor-led foundation section
- 15-question readiness quiz
- game unlock gate at 12/15
- 9 scenes mapped from the crypto investigation workshop
- custom pixel character sprites in `assets/characters.png`
- CC0 prop sprites in `assets/props-modern-houses.png`
- playable top-down movement
- NPC dialogue
- evidence collection
- clickable evidence dossier with detailed case notes
- checkpoint questions mapped to Google Form Q5-Q34
- single choice, multi-select, and short text answer validation
- pre-game field manual / onboarding screen
- bottom-right manual button during play
- room progression locked until checkpoint answers are correct
- smaller decorated RPG rooms per scene
- route progression between scenes with backtracking to previous rooms
- in-game map overlay with `M`

## Asset Notes

Character sprites are custom-made for this prototype. The design direction was researched from license-safe CC0 pixel sprite references, including:

- OpenGameArt: RPG character sprites by GrafxKid
- OpenGameArt: Top down player sprite sheet (Julia) by ArlanTR
- OpenGameArt: Hero character sprite sheet by Fry
- OpenGameArt: "Modern Houses" Tileset TopDown by Ritpop (CC0) for room prop decoration

No third-party character sprite image was copied into this project. Prop decoration uses the CC0 Modern Houses tileset noted above.
